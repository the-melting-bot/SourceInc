# SourceInc

Workspace for the SourceAbility site **rebrand / redesign** (exported from Stitch).

## Next steps

1. Paste your Stitch export here (replace or add `index.html`, assets, CSS, etc.).
2. Open `index.html` in a browser or use a local server (e.g. `npx serve .` from this folder).
3. When you are ready to version it: this folder is set up as its own Git repository (`git init` already run if you used the setup script).

## Relationship to `sourceability-website`

The existing `sourceability-website` project stays as-is. This repo is the fresh redesign track until you choose to merge or replace.
